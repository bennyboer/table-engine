Fixed rows/columns

The Table Engine currently supports fixing rows and columns to the left and top. Fixed rows or columns will always be visible, no matter where the user is scrolling in the table. You can enabled fixed rows or columns by setting engine.getOptions().renderer.view.fixedRows and/or engine.getOptions().renderer.view.fixedColumns to a non-negative number.
