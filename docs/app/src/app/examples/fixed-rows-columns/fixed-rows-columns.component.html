Fixed rows/columns

The Table Engine currently supports fixing rows and columns to the left, top, right and bottom. Fixed rows or columns will always be visible, no matter where the user is scrolling in the table. You can enabled fixed rows or columns by configuring engine.getOptions().renderer.view.fixedAreas to your liking.
