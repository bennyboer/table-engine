6. Borders

If there is a cell model and a selection model, might there even be a border model for managing borders in the table? Yes, absolutely!

What is a border?

A border is a colorful line that is used to visually separate cells. You can choose between different border sizes (or thicknesses), colors and styles (solid, dotted, dashed). What is important: borders only have a visual effect, they do not affect row/column or cell sizes.
